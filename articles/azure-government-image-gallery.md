#  <a name="azure-government-marketplace-images"></a>Azure Government Marketplace-kuvat

## <a name="overview"></a>Yleiskatsaus 
Microsoft Azure Government Marketplace on samanlainen kokemus julkisen Azure-portaaliin.  Asiakkaat voivat valita käyttöön valmiita kuvia Microsoftin ja sen kumppanit tai lataa omia näennäiskiintolevyjen.  Tämä tarjoaa joustavuutta ottamaan standardoitu kuvat tarvittaessa.

Alla on luettelo käytettävissä olevat kuvat sisällä Azure Government Marketplace.  Joissakin valmiin Sisällytä käytön mukaan käyttöoikeudet tiettyyn ohjelmisto.  Tarkista lisäohjeita <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">Virtuaalikoneen hinnat</a> sivu ja käsitellä Microsoft-tilin ryhmäsi tai jälleenmyyjältä, Azure Government hinnat.

### <a name="step-2"></a>Vaihe 2
Käynnistä Lueminut-tiedosto mallin napsauttamalla "käyttöönotto ja Azure Government-painiketta tai linkkiä sivun yläreunassa.

### <a name="step-3"></a>Vaihe 3
Selain-sivu avautuu mallin käyttöönottoa sivu Azure Government-portaalissa. Täytä tarvittavat kentät-parametrit-sivu ja valitse "OK".

### <a name="step-4"></a>Vaihe 4
Hyväksy juridiset ehdot, ja valintamerkki "Raporttinäkymät-ikkunan kiinnittäminen"-ruutuun ja valitse sitten Luo. 

>[AZURE.NOTE] Jotta voit jatkaa on kelvollinen Azure Government-tili. Kohtaat Azure Government potenssiin organisaation tilaa [Azure Government-kokeiluversio](https://azuregov.microsoft.com/trial/azuregovtrial).

### <a name="step-5"></a>Vaihe 5
Näiden suljetaan ja se voi siirtyä takaisin näkyy kuva käyttöönoton portaalin Raporttinäkymät-ikkunan näkymään. Tämä kestää useita minuutteja. Kun AM ratkaisu on käynnissä, se näkyy koontinäytössä. Jos olet valinnut "Raporttinäkymät-ikkunan kiinnittäminen"-vaihtoehto vaiheessa 4, sinun on Etsi-resurssien AM. 

>[AZURE.NOTE] Ratkaisu mallit Microsoftin ja kolmansien osapuolten vain pohjana. Olet yksin vastuussa kehittäminen, käyttöönoton ja hallita sovellusten ja/tai tilaukset, jotka on käytössä Microsoft Azure ja Azure Government lakeja, asetukset ja sopimukseen velvoitteiden mukaisesti.  Nämä mallit "AS-on" ilman mitään minkäänlaista, onko tässä express, epäsuora tai lakisääteisten ja Microsoft nimenomaisesti myönnä kaikki takuita kaupalliseen käytettävyyteen, johonkin tiettyyn tarkoitukseen tai ei.
