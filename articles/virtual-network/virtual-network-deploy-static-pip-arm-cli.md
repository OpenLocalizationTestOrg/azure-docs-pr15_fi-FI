## <a name="step-1---start-your-script"></a>Vaihe 1 – komentosarja

Voit ladata käytetty koko bash komentosarja [tähän](https://raw.githubusercontent.com/Azure/azure-quickstart-templates/master/IaaS-Story/03-Static-public-IP/virtual-network-deploy-static-pip-arm-cli.sh). Noudata seuraavia ohjeita voit muuttaa toimimaan ympäristössäsi komentosarja.

1. Muuttaa arvoja muuttujien, haluat käyttää käyttöönoton arvojen perusteella. Jäljempänä tässä asiakirjassa käytettävästä Yhdistä arvot.

        # Set variables for the new resource group
        rgName="IaaSStory"
        location="westus"
        
        # Set variables for VNet
        vnetName="TestVNet"
        vnetPrefix="192.168.0.0/16"
        subnetName="FrontEnd"
        subnetPrefix="192.168.1.0/24"
        
        # Set variables for storage
        stdStorageAccountName="iaasstorystorage"
        
        # Set variables for VM
        vmSize="Standard_A1"
        diskSize=127
        publisher="Canonical"
        offer="UbuntuServer"
        sku="14.04.2-LTS"
        version="latest"
        vmName="WEB1"
        osDiskName="osdisk"
        nicName="NICWEB1"
        privateIPAddress="192.168.1.101"
        username='adminuser'
        password='adminP@ssw0rd'
        pipName="PIPWEB1"
        dnsName="iaasstoryws1"

## <a name="step-2---create-the-necessary-resources-for-your-vm"></a>Vaihe 2: Luo tarvittavat resurssit oman AM

Ennen kuin luot AM, tarvitset resurssiryhmä, VNet, julkiseen IP ja NIC AM käytettävän.

1. Luo uusi resurssiryhmä.

        azure group create $rgName $location
        
2. Luo VNet ja aliverkon.
        
        azure network vnet create --resource-group $rgName \
            --name $vnetName \
            --address-prefixes $vnetPrefix \
            --location $location
        azure network vnet subnet create --resource-group $rgName \
            --vnet-name $vnetName \
            --name $subnetName \
            --address-prefix $subnetPrefix

3. Luo julkinen IP-resurssi. 

        azure network public-ip create --resource-group $rgName \
            --name $pipName \
            --location $location \
            --allocation-method Static \
            --domain-name-label $dnsName 

4. Luo AM aliverkon luotu edellä julkiseen IP-verkkoliittymän (NIC). Ilmoitus komennot ensimmäisiin käytetään hakemiseen edellä luotu aliverkon **tunnus** .

        subnetId="$(azure network vnet subnet show --resource-group $rgName \
                        --vnet-name $vnetName \
                        --name $subnetName|grep Id)"

        subnetId=${subnetId#*/}
        
        azure network nic create --name $nicName \
            --resource-group $rgName \
            --location $location \
            --private-ip-address $privateIPAddress \
            --subnet-id $subnetId \
            --public-ip-name $pipName

    >[AZURE.TIP] Edellä ensimmäisen komennon käyttää [grep](http://tldp.org/LDP/Bash-Beginners-Guide/html/sect_04_02.html) ja [merkkijonon käytöstä](http://tldp.org/LDP/abs/html/string-manipulation.html) (tarkemmin sanottuna alimerkkijono poistaminen). 

5. Luoda tallennustilan tilin isännöimiseen AM OS-asema.

        azure storage account create $stdStorageAccountName \
            --resource-group $rgName \
            --location $location --type LRS 

## <a name="step-3---create-the-vm"></a>Vaihe 3 – Luo AM 

Nyt kun kaikki tarvittavat resurssit ovat paikallaan, voit luoda uuden AM.

1. Luo AM.

        azure vm create --resource-group $rgName \
            --name $vmName \
            --location $location \
            --vm-size $vmSize \
            --subnet-id $subnetId \
            --nic-names $nicName \
            --os-type linux \
            --image-urn $publisher:$offer:$sku:$version \
            --storage-account-name $stdStorageAccountName \
            --storage-account-container-name vhds \
            --os-disk-vhd $osDiskName.vhd \
            --admin-username $username \
            --admin-password $password

2. Tallenna komentosarjatiedosto.

## <a name="step-4---run-the-script"></a>Vaihe 4 - komentosarjan suorittaminen

        sh myscript.sh
