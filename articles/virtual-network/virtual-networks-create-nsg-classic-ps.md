## <a name="how-to-create-the-nsg-for-the-front-end-subnet"></a>Edusta-aliverkon NSG luominen
Voit luoda NSG nimeltä nimetty **NSG edusta** yllä skenaarion perusteella, noudattamalla seuraavia ohjeita:

1. Jos et ole aikaisemmin käyttänyt PowerShellin Azure-artikkelissa [asentaminen ja määrittäminen PowerShellin Azure](../powershell-install-configure.md) ja noudata ohjeita päässä Azure Kirjaudu ja valitse tilauksen.

3. Luo **NSG FrontEnd**nimeltä verkon käyttöoikeusryhmän.

        New-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" -Location uswest `
            -Label "Front end subnet NSG"

    Odotettu tulos:

        Name         Location   Label               
        ----         --------   -----               
        NSG-FrontEnd West US    Front end subnet NSG


4. Käytön salliminen Internetistä portti 3389 suojauksen säännön luominen

        Get-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" `
        | Set-AzureNetworkSecurityRule -Name rdp-rule `
            -Action Allow -Protocol TCP -Type Inbound -Priority 100 `
            -SourceAddressPrefix Internet  -SourcePortRange '*' `
            -DestinationAddressPrefix '*' -DestinationPortRange '3389'

    Odotettu tulos:

        Name     : NSG-FrontEnd
        Location : Central US
        Label    : Front end subnet NSG
        Rules    :

                      Type: Outbound

                                                           Prefix          Range         Address Prefix   Port Range

                   ALLOW INTERNET       65001     Allow    *               *             INTERNET         *              *       
                   OUTBOUND                                                                                                      
                   DENY ALL OUTBOUND    65500     Deny     *               *             *                *              *

4. Sallitaan Internetistä porttia 80 suojauksen säännön luominen

        | Set-AzureNetworkSecurityRule -Name web-rule `
            -Action Allow -Protocol TCP -Type Inbound -Priority 200 `
            -SourceAddressPrefix Internet  -SourcePortRange '*' `
            -DestinationAddressPrefix '*' -DestinationPortRange '80'

    Odotettu tulos:

## <a name="how-to-create-the-nsg-for-the-back-end-subnet"></a>Uudelleen aliverkon NSG luominen
3. Luo **NSG Taustajärjestelmä**nimeltä verkon käyttöoikeusryhmän.

        New-AzureNetworkSecurityGroup -Name "NSG-BackEnd" -Location uswest `
            -Label "Back end subnet NSG"

    Odotettu tulos:

        Name        Location   Label              
        ----        --------   -----              
        NSG-BackEnd West US    Back end subnet NSG


4. Sallitaan edusta-aliverkosta porttia 1433 (SQL Serverin käyttämä oletusportti) suojaussäännön luominen

        Get-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" `
        | Set-AzureNetworkSecurityRule -Name rdp-rule `
            -Action Allow -Protocol TCP -Type Inbound -Priority 100 `
            -SourceAddressPrefix 192.168.1.0/24  -SourcePortRange '*' `
            -DestinationAddressPrefix '*' -DestinationPortRange '1433'

    Odotettu tulos:

        Name     : NSG-BackEnd
        Location : Central US
        Label    : Back end subnet NSG
        Rules    :

        | Set-AzureNetworkSecurityRule -Name block-internet `
