Tässä artikkelissa on esimerkki kokoonpanon määrittäminen Windows VMs Azure AM tunnisteet.

Lisätietoja tiedostotunniste on artikkelissa [Azure AM laajennukset yleiskatsaus.](virtual-machines-windows-extensions-features.md)

Lisätietoja authoring tunniste mallit-kohdassa [yhtä aikaa muiden kanssa tunniste mallien.](virtual-machines-windows-extensions-authoring-templates.md)

Tässä artikkelissa esitellään odotettu määritysten arvot joissakin Windows-laajennuksia.

## <a name="sample-template-snippet-for-vm-extensions-with-iaas-vms"></a>Esimerkki mallin koodikatkelman AM laajennusten IaaS VMs varten.
Malli-koodikatkelman käyttöönoton laajennukset Näyttää seuraavalla tavalla:

      {
      "type": "Microsoft.Compute/virtualMachines/extensions",
      "name": "MyExtension",
      "apiVersion": "2015-05-01-preview",
      "location": "[parameters('location')]",
      "dependsOn": ["[concat('Microsoft.Compute/virtualMachines/',parameters('vmName'))]"],
      "properties":
      {
      "publisher": "Publisher Namespace",
      "type": "extension Name",
      "typeHandlerVersion": "extension version",
      "autoUpgradeMinorVersion":true,
      "settings": {
      // Extension specific configuration goes in here.
      }
      }
      }

### <a name="customscript-extension-14"></a>CustomScript tunniste 1.4.
      {
          "publisher": "Microsoft.Compute",
          "type": "CustomScriptExtension",
          "typeHandlerVersion": "1.4",
          "settings": {
              "fileUris": [
                  "http: //Yourstorageaccount.blob.core.windows.net/customscriptfiles/start.ps1"
              ],
              "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1"
          },
          "protectedSettings": {
            "storageAccountName": "yourStorageAccountName",
            "storageAccountKey": "yourStorageAccountKey"
          }
      }

#### <a name="parameter-description"></a>Parametrin kuvaus:

- fileUris: pilkku komentoriviin sisältyy luettelo tiedostoista, jotka ladataan AM-tunnisteen mukaan URL-osoitteet. Tiedostoja ladataan, jos mitään ei ole määritetty. Jos tiedostot ovat Azuren tallennustilaan, fileURLs voidaan merkitä yksityinen ja kohdan storageAccountName ja storageAccountKey voi välittää yksityinen parametreiksi näiden tiedostojen käyttäminen.
- commandToExecute: [pakollinen parametri]: Tämä on komento, joka suoritetaan tunnisteen mukaan.
- storageAccountName: [valinnaisten parametrien]: tallennustilan tilin nimi käyttämiseen fileURLs, jos ne ovat merkitä yksityiseksi.
- storageAccountKey: [valinnaisten parametrien]: tallennustilan tilin avain käyttäminen fileURLs, jos ne ovat merkitä yksityiseksi.

### <a name="customscript-extension-17"></a>CustomScript tunniste 1.7.

Tutustu CustomScript versio 1.4 parametrin kuvaus. Version 1.7 esitellään tuki komentosarjan parameters(commandToExecute) lähettämällä sen protectedSettings, jolloin ne salataan ennen lähettämistä. 'commandToExecute' parametri voidaan määrittää asetukset tai protectedSettings mutta ei molemmissa.

        {
            "publisher": "Microsoft.Compute",
            "type": "CustomScriptExtension",
            "typeHandlerVersion": "1.7",
            "settings": {
                "fileUris": [
                    "http: //Yourstorageaccount.blob.core.windows.net/customscriptfiles/start.ps1"
                ],
                "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1"
            },
            "protectedSettings": {
              "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1",
              "storageAccountName": "yourStorageAccountName",
              "storageAccountKey": "yourStorageAccountKey"
            }
        }

          "typeHandlerVersion": "2.1(Recommendation is to use the latest version)",

              "ModulesUrl": "https://UrlToZipContainingConfigurationScript.ps1.zip",

                      "UserName": "UsernameValue2",
                      "Password": "PrivateSettingsRef:Key2"
                  }
              }

          "protectedSettings": {
              "Items": {
                  "Key1": "PasswordValue1",
                  "Key2": "PasswordValue2"

          }
      }

### <a name="azure-diagnostics"></a>Azure diagnostiikka

Lisätietoja diagnostiikka määrittämisestä on artikkelissa [Azure diagnostiikka-tunniste](virtual-machines-windows-extensions-diagnostics-template.md)

          {
            "publisher": "Microsoft.Azure.Diagnostics",
            "type": "IaaSDiagnostics",
            "typeHandlerVersion": "1.5",
            "autoUpgradeMinorVersion": true,
            "settings": {
              "xmlCfg": "[base64(variables('wadcfgx'))]",
              "storageAccount": "[parameters('diagnosticsStorageAccount')]"
            },
            "protectedSettings": {
            "storageAccountName": "[parameters('diagnosticsStorageAccount')]",
            "storageAccountKey": "[listkeys(variables('accountid'), '2015-05-01-preview').key1]",
            "storageAccountEndPoint": "https://core.windows.net"
          }
          }
